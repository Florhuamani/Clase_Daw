Elementos de Input

Inicio Inputs Buttons

Los input son los elementos que se crean como contenido de un formulario, y nos permite interactuar con el usuario, recolectar información a través de una imagen.

Tipos de Input

  • button: botonoes ques se pueden programar para controlar una funcioanlidad personalizada.
  • checkbox: casillas que se marcan al activarse.
  • color: Permite especificarun color, solo se permite colores hexadecimales.
  • date: Crea campos de entrada que permite al usuario introducir una fecha.
  • date-time: controles de entrada que permiten ingresar fecha y hora.
  • file:seleccionar archivos de mi almacenamiento.
  • image: crear botones de envío gráfico, es decir, botones que toman la forma de una imagen en lugar de un texto.
  • month: crea campos de entrada que permite al usuariointroducir el mes y año.
  • number: usado para que el usuario pueda ingresar un número
  • radio: se utilizan generalmente en grupos de radio : colecciones de botones de radio que describen un conjunto de opciones relacionadas.
  • range: Se usa para especificar un valor numérico que no debe ser mayor o menor que el valor dado, esto se representa mediante un control deslizante.
  • reset: se representan como botones, con un click controlador de eventos predeterminado que restablece todas las entrasdas del formulario a sus valores iniciales.
  • search: son campos de texto diseñados para que el usuario introduzca consultas de búsqueda.
  • submit: se representa como botón que al hacer click, intenta enviar el formulario al servidor
  • time: permite ingresar la hora.
  • url: se usa para ingresar y editar una URL.
  • week: permite al usuario ingresar el número de semana.
