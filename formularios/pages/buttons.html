Elementos de buttons

Inicio Inputs Buttons Formulario de Contacto
  • type: Define el tipo de botón (`submit`, `reset`, `button`).
  • disabled: Desactiva el botón, impidiendo la interacción.
  • name : Define un nombre para el botón, útil en formularios.
  • value : Especifica un valor asociado al botón.
  • autofocus: Hace que el botón reciba el foco automáticamente al cargar la página.
  • form: Asocia el botón a un formulario específico mediante su `id`.
  • formaction: Especifica la URL a la que se enviará el formulario cuando se haga clic en el botón.
  • formenctype: Define cómo se codifican los datos del formulario (`application/x-www-form-urlencoded`, `multipart/form-data`, `text/plain`).
  • formmethod: Especifica el método HTTP para enviar el formulario (`GET`, `POST`).
  • formnovalidate: Indica que el formulario no debe validarse al enviarse.
  • formtarget: Define dónde se abrirá el resultado del envío (`_self`, `_blank`, `_parent`, `_top`).
  • class: Permite asignar clases CSS al botón.
  • id : Define un identificador único para el botón.
  • style : Permite aplicar estilos en línea.
  • title : Muestra un texto emergente cuando el usuario pasa el cursor sobre el botón.
